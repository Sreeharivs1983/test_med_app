# StayHealthy - Medical Appointment Booking Website

## Project Overview
This project is a front-end implementation of the StayHealthy Medical Appointment Booking system.

## Features
- User Registration (Sign Up)
- User Login
- Doctor Search
- Appointment Booking
- Consultation Reviews

## Technologies Used
- HTML
- CSS
- Git
- GitHub
- React
- Figma UI/UX
## Notes
This project is part of the IBM Front-End Developer Capstone Practice Project.
